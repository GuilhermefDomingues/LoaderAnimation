# Bouncing Dots Loader

Projeto desenvolvido com HTML e CSS que cria uma animação de carregamento utilizando quatro círculos coloridos. Cada elemento possui um atraso de animação diferente, produzindo um efeito de movimento contínuo e fluido.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- CSS Animations

## ✨ Funcionalidades

- Loader animado
- Movimento vertical alternado
- Círculos coloridos
- Animações suaves
- Sem JavaScript
- Design minimalista

## 📂 Estrutura do Projeto

```bash
Bouncing-Dots-Loader/
│
├── index.html
├── css/
│   └── style.css
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.
